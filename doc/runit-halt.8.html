Name

runit-halt - stop the system

Synopsis

init 0
runit-halt

Description

runit-halt causes the Unix process no 1 runit(8) to halt the system. Normally runit-halt is run by runit-init(8) if called as init 0.

To signal runit(8) the system halt request, runit-halt removes the /etc/runit/reboot file and touches /etc/runit/stopit. Then a CONT signal is sent to runit(8).

Exit Codes

runit-halt always returns 0.

See Also

runit-init(8), runit(8), runit-reboot(8)
