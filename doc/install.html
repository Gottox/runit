runit - installation

Check that you have the recent version of daemontools installed.

Download runit-0.1.2.tar.gz into /package and unpack the archive 

  # cd /package
  # gunzip runit-0.1.2.tar
  # tar -xpf runit-0.1.2.tar
  # rm runit-0.1.2.tar
  # cd admin/runit-0.1.2
Compile and install the runit programs 
  # package/install
gzip the man pages and copy them into a reasonable directory in your $MANPATH, e.g. 
  # mkdir -p /usr/local/man/man8/
  # gzip man/runit.8 && cp -f man/runit.8.gz /usr/local/man/man8/
  # gzip man/runit-init.8 && cp -f man/runit-init.8.gz /usr/local/man/man8/
  # gzip man/runit-halt.8 && cp -f man/runit-halt.8.gz /usr/local/man/man8/
  # gzip man/runit-reboot.8 && cp -f man/runit-reboot.8.gz /usr/local/man/man8/
Refer to replacing init for replacing sysvinit with runit.
