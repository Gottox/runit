runit - installation

Check that you have the recent version of daemontools installed.

Download runit-0.2.7.tar.gz into /package and unpack the archive 

  # cd /package
  # gunzip runit-0.2.7.tar
  # tar -xpf runit-0.2.7.tar
  # rm runit-0.2.7.tar
  # cd admin/runit-0.2.7
Compile and install the runit programs 
  # package/install
If you want to make the man pages available in the /usr/local/man/ hierarchy, do: 
  # package/install-man
Refer to replacing init for replacing init with runit.
