runit - upgrading from previous versions

0.4.1 to 0.5.0

There is a new program: utmpset. This program is used in the getty run scripts to enable local login accounting. You might want to adapt your getty run scripts /service/getty-*/run and stage 3 /etc/runit/3, see the examples in /package/admin/runit/etc/.
